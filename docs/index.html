SITO WEB del gruppo epidemiologico EpiDataItalia

Analisi dei dati della pandemia di COVID-19 in Italia e modelli predittivi

Aggiornamenti automatici:

  • ore ~09:00 Nazioni e Continenti
  • ore ~19:00 Italia e Regioni

EpiDataItalia è un gruppo di studio e ricerca formato da volontari, auto-finanziato e senza fini di lucro, che cerca di offrire analisi e previsioni sulla pandemia di COVID-19, con particolare attenzione sulla situazione in Italia.

Sostieni il nostro lavoro su Patreon o offrici un caffè su buymeacoffee

Grazie per il vostro supporto e interesse.

PROGETTI APERTI:

  • Correlazione tra inquinanti atmosferici e casi di COVID-19 nelle province lomdarde nel 2020

  • Correlazione tra vaccinazioni ed Rt nel mondo per la stima dell'efficacia vaccinale

Per proposte di collaborazione, scrivere a info@epidata.it

I dati italiani sono ricavati dal repository GitHub della Presidenza del Consiglio dei Ministri e Dipartimento di Protezione Civile: Dati COVID-19 Italia.

Per i dati delle altre nazioni è utilizzato il repository GitHub della Johns Hopkins University: COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University.

I sondaggi per la compliance della popolazione sono tratti dal repository GitHub di YouGov: Imperial College London YouGov Covid 19 Behaviour Tracker Data Hub
