Per informazioni info@epidata.it

I dati italiani sono ricavati dal repository GitHub della Presidenza del Consiglio dei Ministri e Dipartimento di Protezione Civile: Dati COVID-19 Italia.

Per i dati delle altre nazioni è utilizzato il repository GitHub della Johns Hopkins University: COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University.

I sondaggi per la compliance della popolazione sono tratti dal repository GitHub di YouGov: Imperial College London YouGov Covid 19 Behaviour Tracker Data Hub
